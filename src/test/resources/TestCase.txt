Тест будет включать: ✅ UI-тестирование (Selenide) – добавление товара в корзину и оформление заказа
✅ API-тестирование (REST Assured) – проверка статуса заказа через API
✅ БД-тестирование (JDBC + PostgreSQL) – проверка сохранения заказа в базе
✅ Эмуляция платежей – тестовая оплата через UI

Тест проверяет, что пользователь может:

Добавить товар в корзину через UI
Оформить заказ через UI (тестовая оплата)
Проверить статус заказа через API
Подключиться к БД и убедиться, что заказ сохранен
